＃Physics Classroom (VR Learning Project)
大学のPBL（Project-based learning）の授業にて制作した、中高生向けのVR物理学習支援アプリです。
1. プロジェクト概要
コンセプト: 「数式を体験に変える」

ターゲット: 物理（力学）の学習に苦手意識を持つ中高生

解決したい課題: 教科書上の2次元的な図解では理解しにくい「3D空間での物体の動き（放物運動など）」を、VR空間で直感的に捉え、数式との繋がりを体感させる。

2. 開発環境・使用技術
Unity: 2022.3.62f3

Language: C#

Platform: Meta Quest 3

Tools: Meta Movement SDK (XR Interaction Toolkit)

3. 主な機能
物理シミュレーションシーン:

  自由落下: 重力加速度による変化を等身大のスケールで体感。

  等速直線運動: 速度の変化と移動距離の関係を可視化。

  放物物運動: 角度や初速を変化させ、軌道の変化をリアルタイムで観察。

インタラクティブ・クイズシステム:

  現象を観察した後、その場に出現するUIでクイズに回答。

シーンマネジメント:

  メインメニューから各学習項目へのスムーズな遷移。
